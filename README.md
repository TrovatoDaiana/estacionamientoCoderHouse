# Sistema de Gestión de Estacionamiento

Este proyecto consiste en el diseño e implementación de una base de datos relacional para la gestión de un sistema de estacionamiento con múltiples sucursales.

El modelo contempla la administración de sucursales, empleados, clientes y vehículos, asegurando la integridad de los datos y evitando inconsistencias como la asignación duplicada de plazas dentro de una misma sucursal.

## Contenido del repositorio

- `trovatoDaianaCoderEstacionamiento.sql`  
  Script SQL de creación de la base de datos, tablas, claves primarias, claves foráneas y restricciones de unicidad.

- `PrimerEntregaEstacionamiento.pdf`  
  Documento explicativo del proyecto que incluye la introducción, objetivos, modelo de negocio, situación problemática, diagrama Entidad–Relación y listado de tablas.

## Tecnologías utilizadas

- MySQL
- SQL
